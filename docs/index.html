fclip

A simple file clipboard system for the terminal that allows you to copy, paste, and move files and directories between different terminal sessions.

ðŸš€ Quick Install

curl -LsSf https://iantimmis.github.io/fclip/install.sh | sh

Commands available after install:

  • fcopy <file> - Copy file/directory to clipboard
  • fpaste - Paste from clipboard
  • fcut <file> - Move file/directory (deletes original on paste)
  • fstatus - Show clipboard contents
  • fclear - Clear clipboard
  • fclip --help - Show help
